KusaReMKN を支えているコンピュータの一覧

kinkan

金柑はミカン科キンカン属に分類される常緑低木の総称である。 果実の大きさは直径 2–3 cm と柑橘類の中で最も小さく、皮ごと食べられることが特徴である。 果皮は甘く、果肉はほどよく酸っぱい。 また、ややほろ苦い後味が残る。 美味しい。

機種 Raspberry Pi 4B V1.2
RAM 4 GB
OS Raspberry Pi OS Lite (bullseye)
用途 Wi-Fi AP, Gateway, DHCP, DNS, FW

ponkan

椪柑はミカン科ミカン属に分類されるインド原産の柑橘類である。 名前のポンはインドの地名プネーに由来する。 果実は蜜柑よりも甘味が強く、独特の甘い香りが特徴である。 果皮はごわごわした手触りだが剥きやすく、内皮も柔らかいため食べやすい。 美味しい。

機種 Raspberry Pi 4B V1.4
RAM 8 GB
OS Raspberry Pi OS Lite (bullseye)
用途 nginx

bontan

文旦はミカン科ミカン属に分類され、木は 3 m ほどまで育つ。 果実の大きさは品種にもよるが、直径 15–25 cm と大きい。 果皮のアルベドの厚さが特徴で、果実の半分を占める程である。 果肉は果汁が少ないが、独特の甘味とさっぱりとした風味を持つ。 美味しい。

機種 Raspberry Pi 4 Model B Rev 1.2
RAM 4 GB
OS FreeBSD 13.2
用途 Apache, 日本語 groff

iyokan

伊予柑はミカン科ミカン属に分類される、日本が原産とされる柑橘類である。 果実は艶のある濃い橙色で、直径は約 8 cm ほどである。 果皮はやや厚めであるが、手で剥くことができる。 果肉は柔らかく果汁を多く含んでおり、甘味酸味のバランスが良い。 美味しい。

機種 Dell XPS 8930
RAM 16 GB
OS Ubuntu 22.04.3 LTS
用途 Vim, YouTube, Twitter

mikan

蜜柑はミカン科ミカン属に分類される常緑低木である。 現代において単に蜜柑といった場合は温州蜜柑を指す。 果実は直径 5–7.5 cm の扁球形で、果皮が柔らかく剥きやすい。 そのまま生食されるほか、缶詰やジュースに加工される。 美味しい。

機種 iPhone Xs
RAM 4 GB
OS iOS 17.0.3
用途 Twitter, YouTube

lemon

檸檬はミカン科ミカン属に分類される。 柑橘類の中でも主に酸味や香りを楽しむ、香酸柑橘類に属する。 果実は紡錘形をしており、熟すと黄色になる。 主に果汁が食用され、その強い酸味と香りからさまざまな料理に添えられる。 美味しい。

機種 SONY VAIO Fit 15
RAM 8 GB
OS FreeBSD 13.2
用途 /usr/src

dekopon

清見と椪柑とを交配して作られた不知火のうち、糖度 13 度以上、酸度 1 度以下を満たすものがデコポンと呼ばれる。 頭部にこぶがあることが特徴で、これをデコと呼ぶ。 果肉は多汁であり、内皮も薄いためそのまま食べやすい。 美味しい。

機種 CHUWI LarkBox X
RAM 12 GB
OS Microsoft Windows 11
用途 持て余している
